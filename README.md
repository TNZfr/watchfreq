# watchfreq
Processor frequency monitoring

# Install
Just copy binary file in your $HOME/bin for example or compile it !<br>
<br>
# Run
Syntax :
watchfreq NbSec<br>
        NbSec : Monitoring time in seconds.<br>
<br>
Display on a Ryzen 9 5950X<br>
